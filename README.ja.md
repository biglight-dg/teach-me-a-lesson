# teach-me-a-lesson — AI・プログラミング初心者のための、やさしく根気強い先生 skill

Claude や AI ツール、プログラミング/IT の概念を、**まったくの初心者や非エンジニアに対して、やさしい先生のように**説明し、作業中もそばで助け続ける [Claude スキル](https://docs.claude.com/en/docs/claude-code/skills)です。

ただ答えを渡すのではなく、ひとつの良いたとえで**「ああ、*そういうことか*！」**という瞬間をつくることに重点を置いています。

## できること

6 つのモードで動きます（普通に頼んでも、スラッシュコマンドで呼んでも OK）。

| モード | コマンド | 内容 |
|--------|----------|------|
| **explain** | `/tl-explain <トピック>` | 概念をたとえ・表・要点で分かりやすく説明（デフォルト） |
| **mistake** | `/tl-mistake` | 何が・なぜ間違ったのかを責めずに診断し、直し方まで案内 |
| **bad-habit** | `/tl-bad-habit` | 繰り返している悪い癖を見つけ、より良いやり方を提案 |
| **knowledge** | `/tl-knowledge <トピック>` | 学んだことを知識ファイル（markdown ノート）として保存 |
| **token** | `/tl-token` | 使い方を観察してトークン/コンテキストの無駄を指摘し、具体的な改善策を提示 |
| **config** | `/tl-config` | 先生の言語・丁寧さ・トーン/人物像・絵文字の量・呼びかけ方を設定 |

スキルが読み込まれている間は、**そばに常駐する助っ人**としても働きます。危険な操作（出所不明のコマンドの貼り付け、API キーの露出、取り消せない操作）、誤解にもとづく依頼、遠回りな非効率なやり方を見つけると、**そっと指摘**し——しつこくせず——判断はあなたに委ねます。

> 注意：この常時サポートはスキルが読み込まれている間（会話の文脈内）に働きます。入力のたびに自動チェックするフック（hook）ではありません。

### 発動の例

- 「MCP が何か、すごく簡単に教えて。完全な初心者なんだ」
- 「API と SDK がいつも混ざる。違いをたとえで説明して」
- 「ターミナルが怖いんだけど、本当に使わないとダメ？」
- 「.env を GitHub に上げちゃった気がするんだけど、何が問題？」 → mistake
- 「私の悪い癖を指摘して」 → bad-habit
- 「トークンを無駄づかいしてる気がする。ちょっと見てくれる？」 → token

## インストール

### 1）スキルフォルダをコピー

```bash
# macOS / Linux
cp -r teach-me-a-lesson ~/.claude/skills/

# Windows (PowerShell)
Copy-Item -Recurse teach-me-a-lesson $env:USERPROFILE\.claude\skills\
```

### 2）スラッシュコマンドをインストール（任意、/tl-* を使う場合）

スキル内の `commands/` フォルダのファイルを、あなたのコマンドフォルダにコピーします。

```bash
# macOS / Linux
cp teach-me-a-lesson/commands/tl-*.md ~/.claude/commands/

# Windows (PowerShell)
Copy-Item teach-me-a-lesson\commands\tl-*.md $env:USERPROFILE\.claude\commands\
```

コマンドがなくても、「これ簡単に説明して」のような言葉で自動的に発動します。コマンドは特定のモードを明示的に呼び出す便利な手段にすぎません。

## （任意）自分の知識ファイルをつなぐ

**知識ファイル**（markdown ノートのフォルダ）があれば、スキルはそれを優先的な根拠として使います。なければ Claude 自身の知識にフォールバックするので、**知識ファイルがまったくなくても問題なく動きます**。

- 既定の検索/保存フォルダ：作業ディレクトリの `./knowledge/`、または `~/.claude/teach-me-a-lesson/knowledge/`
- すでにノートフォルダがあれば、そこを指定するだけです。
- 詳しいルールと任意のインデックス形式は `references/knowledge-files.md` を参照。

> このリポジトリには個人の知識ドキュメントは**一切含まれていません**。公開しているのは教え方の方法論（スキル本体）だけです。

## トーンと言語の設定

先生の**言語・丁寧さ・雰囲気**を好みに合わせて調整できます。たとえば韓国語では、敬語/タメ口（존댓말/반말）だけでなく、**雰囲気**（あたたかい・淡白・元気・かっちり）まで調整します。

- `/tl-config` — 言語 → 丁寧さ → トーン/人物像 → 絵文字の量 → 呼びかけ方、の順に尋ねて保存します
- 自然な言い方でも OK ——「タメ口で元気に」「英語で説明して」「絵文字を減らして」
- 設定は `~/.claude/teach-me-a-lesson/config.json` に保存されます（個人のトーンファイルなので、配布物には**含まれません**）。設定がなければ**既定値（あたたかい先生、丁寧な韓国語 해요체）**で動きます。
- どんなトーンでも、教える質（たとえ・責めない・詰め込みすぎない・出典明記）は変わりません。`references/tone-config.md` を参照。
